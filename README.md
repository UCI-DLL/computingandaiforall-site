# Computing and AI for All — homepage redesign prototype

This static prototype restyles the current Computing and AI for All homepage using the shared visual language of the Digital Learning Lab and CAIforAll curriculum sites. It preserves the original homepage content, images, video, navigation, and section order without adding or removing material.

## Preview

Serve this folder with any local static server, then open `index.html` in a browser.

The prototype keeps navigation links pointed at the current public website while the redesign is under review.
